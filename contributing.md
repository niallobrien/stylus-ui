## Getting Started

1. Install dependencies

 * Git (<http://git-scm.com>)
 * Gulp (<http://gulpjs.com>)
 * Node.js (<http://nodejs.org>)
 * NPM (<http://npmjs.org>)
 * Stylus (<http://stylus-lang.com>)

1. Clone repository

   ```bash
   $ git clone git@github.com:niallobrien/stylus-ui.git
   $ cd stylus-ui
   ```

1. Install node dependencies using npm

   ```bash
   $ npm install
   ```

1. Run

   ```bash
   $ npm start
   ```
